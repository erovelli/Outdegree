# Browsing Graph

A Chrome extension that records your web navigations as a directed graph you can
explore over time. **100% local · no network · open source.**

- [Privacy Policy](privacy-policy.html)
- [Source code on GitHub](https://github.com/erovelli/Outdegree)

Your browsing data is stored only on your own device, in your browser's local
storage. The extension makes no network requests — enforced by the browser
(`host_permissions: []` and CSP `connect-src 'none'`), not merely promised.
