# Support

Outdegree is an open-source portfolio project maintained on a best-effort,
unpaid basis. Here's where to go:

- **Found a bug?** Open a [bug report](https://github.com/erovelli/Outdegree/issues/new?template=bug_report.yml).
  Please **do not** attach exported browsing data — describe the steps instead.
- **Have an idea?** Open a [feature request](https://github.com/erovelli/Outdegree/issues/new?template=feature_request.yml).
  Note that proposals must preserve the local-only / no-network guarantees.
- **Question or discussion?** Use
  [GitHub Discussions](https://github.com/erovelli/Outdegree/discussions) if
  enabled; otherwise a regular issue is fine.
- **Privacy questions?** See the [Privacy Policy](https://erovelli.github.io/Outdegree/privacy-policy.html)
  and [SECURITY.md](SECURITY.md).
- **Security vulnerability?** Follow [SECURITY.md](SECURITY.md) — report it
  privately, not in a public issue.

Before filing, a quick check of the [README](README.md) and existing issues often
saves time. Thanks for using Outdegree!
